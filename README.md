# Tomas Montero | Desarrollo de Software

Sitio web profesional y portafolio interactivo para **Tomas Montero**, enfocado en ofrecer soluciones tecnológicas que impulsan la conversión y el alcance de negocios digitales.

## 🌟 Características Principales

- **Diseño Premium e Interactivo**: Estética profesional utilizando una paleta de colores Navy & Amber diseñada para transmitir confianza y autoridad.
- **Mockups 3D de Notebooks**: Galería de proyectos con visualizaciones interactivas que reaccionan al mouse y al tacto, proporcionando una experiencia envolvente.
- **Optimización UX/UI**: Implementación de leyes de diseño (Focal Point, Affordance) para guiar al usuario hacia la conversión.
- **Formulario de Contacto Inteligente**: Integración con API de correo y acceso directo a WhatsApp para una comunicación fluida.
- **Totalmente Responsivo**: Experiencia optimizada para dispositivos móviles, tablets y escritorio.

## 🛠️ Stack Tecnológico

- **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)
- **Lenguaje**: [TypeScript](https://www.typescriptlang.org/)
- **Estilos**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animaciones**: [Framer Motion](https://www.framer.com/motion/)
- **Iconos**: [Lucide React](https://lucide.dev/)
- **Backend Email**: [Nodemailer](https://nodemailer.com/)

## 🎨 Sistema de Diseño (Navy & Amber)

El sitio utiliza un sistema de diseño propio documentado para maximizar la jerarquía visual:

- **Navy (#0D1F3C)**: Color primario para confianza y solidez institucional.
- **Amber (#D97706)**: Color de contraste para Call-to-Action (CTA), diseñado para atrapar el ojo.
- **Steel Blue**: Colores de soporte para profundidad visual.

## 🚀 Instalación y Desarrollo

1. **Clonar el repositorio**:

   ```bash
   git clone https://github.com/Monterolautaro/tomasmontero-software.git
   ```

2. **Instalar dependencias**:

   ```bash
   npm install
   ```

3. **Configurar variables de entorno**:
   Crea un archivo `.env` basado en `.env.example` con tus credenciales de SMTP para el formulario.

4. **Iniciar servidor de desarrollo**:

   ```bash
   npm run dev
   ```

5. **Abrir en el navegador**:
   Visita [http://localhost:3000](http://localhost:3000)
